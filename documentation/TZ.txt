web_ui.py:

load_user - авторизация пользователя на сайте, возвращает user_id

theme_check - получение темы для страницы, если пользователь авторизован - возвращается значение из базы данных, если нет то возвращается dark

translate_list - перевод ответа или запроса для AI Dungeon в виде списка, получает список не переведённого текста и исходный язык, возвращает переведённый текст 
 
translate - перевод ответа или запроса для AI Dungeon в виде строки, получает не переведённый текст и исходный язык, возвращает переведённый текст

check_file - проверяет, есть ли изменения в файле. Файл свежий в том случае, если содержит подтверждающий это атрибут.
Если атрибут говорит, что файл обновлён, то возвращает истину и отмечает файл как 'не новый', а также извлекает содержимое.
Если файл уже отмечен несвежим, то вернёт ложь.

write_prompt - запись сообщения пользователя в txt файл и отмечается что он обновлён

default_page - главная страница, обработчик адреса "/"

chat_page - страница с игрой, обработчик адреса "/chat"

settings_page - страница с настройками, обработчик адреса "/settings"

load_page - страница для загрузки своего сценария игры, обработчик адреса "/load"

register_page - страница для регистрации, обработчик адреса "/register"

login_page - страница для авторизации, обработчик адреса "/login"

logout - выход пользователя

main - запуск сайта и создание сессии базы данных
